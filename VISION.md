# SermonPrint Vision

SermonPrint exists to help pastors write, prepare, export, and preach sermon manuscripts without fighting their software.

## Principles

1. Pastor first.
2. Clean Markdown underneath.
3. Word-like manuscript experience.
4. What you see is what you preach.
5. One command: SermonPrint: Edit & Export.

## v0.5 Goal

Build a real manuscript layout foundation. Page size, margins, spacing, typography, visual page guides, PDF export, and booklet export must all come from the same layout settings.
